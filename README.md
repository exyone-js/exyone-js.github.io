# Exyone Blog

Exyone's Home Page

Powered by [Eleventy](https://www.11ty.dev/)

## 特性

- **干净简洁**：三栏式布局，经典简约卡片设计
- **暗色模式**：跟随系统偏好自动切换深色/浅色主题
- **响应式**：桌面三栏 → 平板两栏 → 移动端单栏适配

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发（热重载）
npm run serve

# 构建生产产物
npm run build
```

构建产物输出到 `_site/` 目录。

## 项目结构

```
_data/           # 全局数据文件
_includes/       # 模板组件
_layouts/        # 布局模板
_posts/          # 博客文章
pages/           # 独立页面
assets/          # 静态资源
  ├── css/         # 样式文件
  ├── js/          # JavaScript
  └── images/      # 图片
media/           # 文章配图
```
